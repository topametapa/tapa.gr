Το content είναι ο βασιλιάς, αλλά χωρίς σωστό technical SEO, ο βασιλιάς κάθεται σε θρόνο που κανείς δεν βλέπει.

Αυτό το checklist περιέχει **τα 15 κρίσιμα σημεία** που πρέπει να τσεκάρεις για να βεβαιωθείς ότι η Google—και οι πελάτες σου—βρίσκουν το site σου.

## Μέρος 1: Core Web Vitals

Τα metrics που η Google χρησιμοποιεί επίσημα ως ranking factors:

### ☐ LCP (Largest Contentful Paint) < 2.5s
Πόσο γρήγορα φορτώνει το κύριο περιεχόμενο. **Target: < 1.2s για competitive edge.**

**Πώς να το βελτιώσεις:**
- Optimize images (WebP, lazy loading)
- Eliminate render-blocking resources
- Use CDN

### ☐ INP (Interaction to Next Paint) < 200ms
Πόσο responsive είναι η σελίδα στα κλικ.

**Πώς να το βελτιώσεις:**
- Minimize JavaScript
- Break up long tasks
- Optimize event handlers

### ☐ CLS (Cumulative Layout Shift) < 0.1
Πόσο σταθερό είναι το layout (τίποτα δεν "πηδάει").

**Πώς να το βελτιώσεις:**
- Set image dimensions
- Reserve space for ads/embeds
- Avoid inserting content above existing content

## Μέρος 2: Crawlability

### ☐ robots.txt Properly Configured
Βεβαιώσου ότι δεν μπλοκάρεις σημαντικές σελίδες.

### ☐ XML Sitemap Submitted
Στείλε το sitemap στο Google Search Console. Update αυτόματα με κάθε νέα σελίδα.

### ☐ No Orphan Pages
Κάθε σελίδα πρέπει να έχει τουλάχιστον ένα internal link που δείχνει σε αυτήν.

### ☐ Logical Internal Linking
Σύνδεσε related content. Βοηθά τη Google να καταλάβει τη δομή του site.

## Μέρος 3: On-Page Elements

### ☐ Unique Title Tags
Κάθε σελίδα: unique, < 60 χαρακτήρες, keyword στην αρχή.

### ☐ Meta Descriptions
< 155 χαρακτήρες, compelling copy, keyword included. Δεν είναι ranking factor αλλά επηρεάζει CTR.

### ☐ Heading Hierarchy
H1 → H2 → H3. Ένα H1 ανά σελίδα. Logical structure.

### ☐ Image Alt Attributes
Περιγραφή της εικόνας. Βοηθά accessibility + image search.

### ☐ Canonical URLs
`<link rel="canonical">` σε κάθε σελίδα. Αποφυγή duplicate content issues.

## Μέρος 4: Structured Data

### ☐ Organization Schema
Βασικές πληροφορίες για την επιχείρηση.

### ☐ LocalBusiness Schema
Διεύθυνση, τηλέφωνο, ωράριο. Essential για τοπικό SEO.

### ☐ BreadcrumbList
Εμφάνιση breadcrumbs στα SERPs.

### ☐ Article/BlogPosting
Για blog posts—author, date, publisher info.

### ☐ FAQ Schema
Εμφάνιση FAQ directly στα search results.

## Μέρος 5: Security & Mobile

### ☐ HTTPS Everywhere
Όχι mixed content. Valid SSL.

### ☐ Mobile-First Design
Responsive, touch-friendly, readable.

### ☐ hreflang Tags
Για ελληνικό content: `hreflang="el"` + `hreflang="x-default"`
